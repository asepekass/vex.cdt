# VEX.CDT (Contract Development Toolkit)
## Version : 1.5.0

### Guided Installation (Building from Scratch)
```sh
$ git clone --recursive https://github.com/vexanium/vex.cdt
$ cd vex.cdt
$ ./build.sh
$ sudo ./install.sh
```

### Installed Tools
---
* eosio-cpp
* eosio-cc
* eosio-ld
* eosio-init
* eosio-abigen
* eosio-abidiff
* eosio-pp (post processing pass for WASM, automatically runs with eosio-cpp and eosio-ld)
* eosio-wasm2wast
* eosio-wast2wasm
* eosio-ranlib
* eosio-ar
* eosio-objdump
* eosio-readelf
